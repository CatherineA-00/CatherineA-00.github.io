Lithographic Print Project

The lithographic print project came with its own set of challenegs. My goal was to make a bookmark for elementary school kids using one color. The easy part was coming up with ideas, the hard part was trying to exicute it using one color. I had made two bookmarks, one that looked like a bumblebee and the spider one I have shown to the right. The spider bookmark is the one I ended up going with, I loved the dropshadow along the superhero text, and its simple design make sense for only one color. The Printing proccess for it was easy, though it helped I had already used a lithographic print press, the only thing new about it was that I never used that specific machine.
